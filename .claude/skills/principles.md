# SpoolmanDB UI Upgrade Principles

1. **Data First** — ตารางและข้อมูลต้องอ่านง่ายและสวยงามก่อนสิ่งอื่นใด
2. **Progressive Enhancement** — เริ่มจากฟังก์ชันการใช้งานหลักให้มั่นคง แล้วจึงเพิ่มเติมเอฟเฟกต์ตกแต่งต่างๆ
3. **Consistency over Creativity** — ยึดการใช้งาน Design Tokens (ตัวแปร CSS) เดียวกันอย่างสม่ำเสมอทุกจุด
4. **Maker Friendly** — ใช้โทนสีที่อบอุ่น (ทอง-ส้ม) แทนการใช้ดีไซน์แนวไซไฟเทคโนโลยีที่จัดจ้านเกินไป
5. **Performance Matters** — หลีกเลี่ยงแอนิเมชันหรือกระจกเบลอ (Glassmorphism) ที่ซับซ้อนเกินไปอันจะส่งผลต่อความเร็วการโหลดเว็บ
6. **Mobile is not afterthought** — ตารางและฟิลเตอร์ต้องยืดหยุ่นและใช้งานได้อย่างสวยงามบนโทรศัพท์มือถือ
