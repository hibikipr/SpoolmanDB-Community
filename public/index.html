Contributor helper

Edit source files in filaments/*.json. Do not hand-edit generated filaments.json.

  1. Step 1 Pick the source file

    Work in filaments/ and keep one manufacturer or focused fix per PR.

  2. Step 2 Edit source JSON

    Add variants, correct fields, and keep generated files out of manual edits.

  3. Step 3 Validate locally

    Run the compiler and schema checks before opening the pull request.

  4. Step 4 Open a PR

    Include product pages, datasheets, or other evidence for the data change.

Local commands

python scripts/compile_filaments.py
check-jsonschema --schemafile materials.schema.json materials.json
check-jsonschema --schemafile filaments.schema.json filaments/*

If the validator is missing, install it with python -m pip install check-jsonschema.

PR checklist

- [ ] Edited source files in filaments/ only, not generated filaments.json
- [ ] Kept the PR focused to one manufacturer or correction set
- [ ] Linked product pages, datasheets, or other evidence
- [ ] Ran python scripts/compile_filaments.py
- [ ] Ran check-jsonschema for materials.json and filaments/*
